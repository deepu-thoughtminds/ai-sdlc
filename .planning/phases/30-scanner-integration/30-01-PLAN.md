## Tasks

### Task 1: Create `backend/services/sonar_scanner.py`

**Files:** `backend/services/sonar_scanner.py`, `backend/tests/test_sonar_scanner.py`
**Action:** create

<action>
Create `backend/services/sonar_scanner.py`. Module-level docstring: "sonar-scanner CLI invocation and CE task polling — SCAN-01, SCAN-02, SCAN-03, SCAN-04."

Imports: `logging`, `os`, `pathlib`, `time`, `httpx`, and `from services.test_executor import TestResult, ToolchainCommand, run_command`.

Module constant: `_CE_POLL_INTERVAL = 5` (seconds between CE task polls).

**Function `_read_ce_task_id(workspace_path: str) -> str | None`:**
Build a `pathlib.Path` to `<workspace_path>/.scannerwork/report-task.txt`. Return `None` if the file does not exist. Read its text (encoding="utf-8"), iterate lines, and return the value after the first `=` on the line starting with `ceTaskId=`. Return `None` if no such line is found.

**Function `_poll_ce_task(sonar_url: str, task_id: str, token: str, timeout_secs: int) -> str`:**
Compute a deadline via `time.monotonic() + timeout_secs`. Loop while `time.monotonic() < deadline`. On each iteration: issue `httpx.get(f"{sonar_url}/api/ce/task", params={"id": task_id}, headers={"Authorization": f"Bearer {token}"}, timeout=10.0)`. If `status_code == 200`, extract `r.json().get("task", {}).get("status", "")`. If status is one of `"SUCCESS"`, `"FAILED"`, `"CANCELLED"`, return it immediately. Catch `httpx.RequestError` and continue (SonarQube momentarily unreachable). Sleep `_CE_POLL_INTERVAL` seconds at end of each iteration. After the loop exits, return `"TIMEOUT"`.

**Function `run_sonar_scan(workspace_path: str, project_key: str, sonar_url: str, token: str, compose_network: str, timeout_secs: int = 300) -> TestResult`:**

Docstring: "Run sonar-scanner-cli and poll CE task to completion. Never raises. All failure paths return TestResult with non-zero returncode."

Derive `project_name` as `project_key.replace("__", "/", 1)` — converts `"acme__my-app"` to `"acme/my-app"` for the human-readable project name property.

Build a `ToolchainCommand` with `name="sonar-scanner"` and command list:
`["docker", "run", "--rm", "--network", compose_network, "-v", f"{workspace_path}:/usr/src", "-e", f"SONAR_HOST_URL={sonar_url}", "-e", f"SONAR_TOKEN={token}", "sonarsource/sonar-scanner-cli:5", f"-Dsonar.projectKey={project_key}", f"-Dsonar.projectName={project_name}", "-Dsonar.sources=/usr/src", "-Dsonar.scm.disabled=true"]`

Call `run_command(cmd, timeout=timeout_secs)` and assign to `result`.

If `result.returncode != 0`, return `result` immediately (scanner exited non-zero; report-task.txt absent; CE polling skipped).

Call `_read_ce_task_id(workspace_path)`. If it returns `None`, return a `TestResult` with `tool="sonar-scanner"`, `returncode=1`, `stdout=result.stdout`, `stderr="Scanner exited 0 but .scannerwork/report-task.txt not found"`, `timed_out=False`.

Call `_poll_ce_task(sonar_url, task_id, token, timeout_secs)` and assign to `ce_status`.

Return a `TestResult(tool="sonar-scanner", timed_out=False)` with:
- `ce_status == "SUCCESS"`: `returncode=0`, `stderr=f"CE task {task_id}: SUCCESS"`
- `ce_status == "TIMEOUT"`: `returncode=1`, `timed_out=True`, `stderr=f"CE task {task_id} polling timed out after {timeout_secs}s"`
- anything else ("FAILED", "CANCELLED"): `returncode=1`, `stderr=f"CE task {task_id}: {ce_status}"`

In all return paths, pass `stdout=result.stdout`.
