# AI-SDLC

Agentic AI platform that embeds a Hermes agent into Jira comment history to automate SDLC stages.

## Setup

### 1. Configure environment

Copy `.env.example` to `.env` and fill in your values. At minimum:

```bash
# Generate a secure JWT secret
echo "JWT_SECRET_KEY=$(python3 -c 'import secrets; print(secrets.token_hex(32))')" >> .env
```

### 2. Start services

```bash
docker compose up -d
```

### 3. Generate a frontend API token

```bash
docker compose exec backend python generate_token.py
```

Add the output token to `.env`:

```
NEXT_PUBLIC_API_TOKEN=<token>
```

### 4. Rebuild and restart the frontend

Next.js inlines `NEXT_PUBLIC_*` variables at build time, so a rebuild is required after changing the token:

```bash
docker compose build frontend && docker compose up -d frontend
```

### 5. Open the app

```
http://localhost:3000/dashboard
```

## Notes

- Re-run steps 3–4 any time you rotate `JWT_SECRET_KEY` or need a new frontend token.
- The token generated by `generate_token.py` is long-lived (10 years) and intended for service-to-service auth between the Next.js frontend and FastAPI backend.
